<h1 align="center">"Server Log"</h1>

<p align="center">마인크래프트 서버 관리를 위한 서버 로그 플러그인</p>
<p align="center">필터링이 없는 서버로그, 이제 안녕.<br>
각종 이벤트를 감지한 로그를 별도로 분류하여 기록됩니다.</p>
<br>
<br>
<br>
<h3 align="left">무엇을 기록하나요?</h3>
<p align="left">기본적으로 년, 월, 일, 몇시를 기록하게 됩니다.<br>
<p align="left">블럭 - 설치, 파괴<br>
<p align="left">채팅 - 닉네임, 채팅 내용<br>
<p align="left">청크 - Config 파일에 설정된 시간(기본값 5분)마다 서버에 로드된 맵 별로 RAM에 적재된 청크 갯수<br>
<p align="left">명령어 - 닉네임, 명령어, 어느 맵인지, 위치 좌표<br>
<p align="left">엔티티 - 서버에 로드된 맵 별로 로드된 엔티티 갯수<br>
<p align="left">아이템 - 닉네임, 무슨 행동(뱉기, 소환, 얻기), 어느 맵인지, 위치 좌표, 무슨 아이템, 수량<br>
<p align="left">플레이어 - 사망, 게임모드 변경시점, 접속, 추방, 로그아웃, 텔레포트<br>
<p align="left">플레이어수 - Config 파일에 설정된 시간(기본값 5분)마다 서버에 로드된 맵 별로 유저 접속 인원<br></p>
